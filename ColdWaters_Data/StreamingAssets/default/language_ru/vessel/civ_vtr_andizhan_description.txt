History=Универсальный сухогруз тип «Андижан». Строились для флота СССР двумя сериями на верфях ГДР в 1957-62гг. Интенсивно использовались во время холодной войны для доставки военных грузов союзникам СССР. Некоторое количество этих судов по-прежнему находятся в составе торгового флота.
