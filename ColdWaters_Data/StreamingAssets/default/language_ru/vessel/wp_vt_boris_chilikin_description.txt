Description=Танкер пр. 1559-В, тип «Борис Чиликин»
History=Танкеры-заправщики проекта 1559-В, проектный шифр «Морской простор», тип «Борис Чиликин» — серия из шести больших морских танкеров (фактически — танкеров комплексного снабжения), строившихся в 1970-х годах для Военно-Морского Флота СССР. Представляют собой вариант гражданского танкера типа «Великий Октябрь», которые строились для советского торгового флота и на экспорт. Они стали первыми океанскими судами снабжения в ВМФ СССР, способными передавать не только топливо, но и другие грузы. Грузовместимость танкера этого типа составляет 13000 т. котельного и дизельного топлива, 500 т. пресной воды, 400 т. продовольствия, 400 т. боеприпасов и 400 т. других предметов МТО.
