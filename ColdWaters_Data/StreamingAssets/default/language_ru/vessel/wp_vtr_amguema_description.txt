Description=Военный транспорт типа «Амгуэма»
History=Ледоколы проекта 550 типа «Амгуэма» — серия советских транспортных ледокольных дизель-электроходов. Суда спроектированы ЦКБ «Айсберг» для работы в северных широтах. Входят в состав гражданского флота, но с началом боевых действий были переоборудованы в военные транспорты. Грузоподъёмность каждого из них составляет солидные 6600 тонн.
