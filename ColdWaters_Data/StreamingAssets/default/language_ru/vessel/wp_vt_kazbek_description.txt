Description=Танкер пр. 563, тип «Казбек»
History=Первый послевоенный проект танкера в СССР. Более 60 кораблей этого типа были построены в 50-х годах для коммерческого судоходства. В 60-х часть из них была передана ВМФ для использования в качестве судов поддержки. Помимо заправки кораблей в море, также способны осущестлять перегрузку сухих грузов, для чего оснащены большой двуногой мачтой и дополнительными кранами.
