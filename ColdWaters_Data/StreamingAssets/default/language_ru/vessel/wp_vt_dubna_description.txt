Description=Танкер типа «Дубна» (1984г)
History=Танкеры типа «Дубна» — серия из четырёх средних морских танкеров, строившихся в 1970-х годах в Финляндии для Военно-Морского Флота СССР. Относится к морским судам обеспечения ВМФ. Служат для обеспечения кораблей и подводных лодок дизельным топливом, мазутом, авиационным керосином, моторным маслом, водой и продовольствием.
